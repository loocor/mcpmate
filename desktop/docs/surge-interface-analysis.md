## 🎯 **核心概念识别**

### **网络代理概念**
1. **代理服务器** - 网络流量的中转节点
2. **出站模式** - 流量处理策略（直连、代理、规则）
3. **策略组** - 代理服务器的逻辑分组
4. **规则匹配** - 流量路由的判断逻辑

### **监控和统计概念**
1. **活动监控** - 实时网络活动展示
2. **流量统计** - 数据传输量的统计分析
3. **请求记录** - HTTP/HTTPS 请求的详细日志
4. **性能指标** - 延迟、速度等性能数据

### **配置管理概念**
1. **配置文件** - 完整的代理配置集合
2. **模块系统** - 功能扩展和插件管理
3. **规则集** - 流量处理规则的集合
4. **设备管理** - 客户端设备的配置和状态

### **系统集成概念**
1. **托盘集成** - 系统托盘的快速访问
2. **全局代理** - 系统级别的代理设置
3. **证书管理** - HTTPS 解密的证书处理
4. **进程监控** - 应用程序级别的网络监控

## 🔍 **界面元素分析**

### **导航模式**
- **侧边栏导航** - 功能模块的主要导航方式
- **分层结构** - More 下的二级功能分组
- **图标 + 文字** - 清晰的功能标识

### **数据展示模式**
- **列表视图** - 进程、请求、规则等数据的展示
- **卡片视图** - 概览信息的模块化展示
- **图表视图** - 流量统计的可视化展示
- **实时更新** - 活动监控的动态数据展示

### **交互模式**
- **点击选择** - 主要的交互方式
- **右键菜单** - 上下文操作
- **开关控制** - 功能启用/禁用
- **下拉选择** - 模式和配置的切换

### **状态指示**
- **颜色编码** - 不同状态的视觉区分
- **图标状态** - 功能启用状态的图标化表示
- **文字描述** - 详细的状态信息说明

## 🎨 **设计语言分析**

### **视觉风格**
- **macOS 原生设计** - 完全符合 macOS 设计规范
- **简洁现代** - 清晰的信息层次和视觉焦点
- **功能导向** - 界面服务于功能，不过度装饰

### **色彩系统**
- **系统色彩** - 使用 macOS 系统颜色
- **状态色彩** - 绿色（正常）、红色（错误）、橙色（警告）
- **中性色彩** - 灰色系的信息展示

### **交互反馈**
- **悬停效果** - 按钮和列表项的悬停状态
- **选中状态** - 清晰的选中视觉反馈
- **加载状态** - 数据加载的进度指示

### **信息密度**
- **适中密度** - 信息丰富但不拥挤
- **分层展示** - 主要信息突出，次要信息收起
- **渐进披露** - 复杂功能的分步展示

## 🔄 **交互流程梳理**

### **用户典型操作流程**

#### **流程1：初次使用配置**
```
1. 启动应用 → 2. 导入/创建配置文件 → 3. 配置代理服务器 → 4. 设置规则 → 5. 启用代理
   ↓
   Overview查看状态 → Policy配置策略组 → Rule设置路由规则 → 托盘启用
```

#### **流程2：日常使用监控**
```
1. 托盘快速切换模式 → 2. Activity监控流量 → 3. Process查看应用 → 4. Dashboard分析统计
   ↓
   快速操作 → 实时监控 → 应用级控制 → 数据分析
```

#### **流程3：问题排查调试**
```
1. Activity发现异常 → 2. Process定位应用 → 3. Rule检查规则 → 4. HTTP Capture抓包分析
   ↓
   异常发现 → 问题定位 → 规则验证 → 深度分析
```

#### **流程4：配置管理维护**
```
1. Profile管理配置 → 2. Module安装插件 → 3. Policy调整策略 → 4. Rule优化规则
   ↓
   配置切换 → 功能扩展 → 策略优化 → 规则调优
```

### **概念关联关系图**
```
配置文件 (Profile)
    ├── 代理策略 (Policy)
    │   ├── 代理服务器组
    │   ├── 负载均衡规则
    │   └── 健康检查设置
    ├── 路由规则 (Rule)
    │   ├── 域名匹配
    │   ├── IP地址匹配
    │   └── 应用程序匹配
    ├── 功能模块 (Module)
    │   ├── 脚本扩展
    │   ├── 广告拦截
    │   └── 数据处理
    └── 系统设置
        ├── HTTPS解密
        ├── HTTP捕获
        └── 外观配置

运行时状态
    ├── 活动监控 (Activity)
    │   ├── 实时连接
    │   ├── 流量统计
    │   └── 地理分布
    ├── 进程管理 (Process)
    │   ├── 应用列表
    │   ├── 流量分配
    │   └── 规则匹配
    └── 仪表板 (Dashboard)
        ├── DNS查询
        ├── 请求日志
        └── 统计图表
```

### **界面导航逻辑**
```
主窗口导航：
├── 监控类 (实时数据)
│   ├── Activity - 网络活动实时监控
│   ├── Overview - 系统状态总览
│   └── Process - 应用程序监控
├── 配置类 (静态设置)
│   ├── Policy - 代理策略配置
│   ├── Rule - 路由规则管理
│   └── More/Profile - 配置文件管理
├── 调试类 (开发工具)
│   ├── HTTP Capture - 请求捕获
│   ├── HTTPS Decryption - 证书管理
│   └── Rewrite & Mapping - 请求重写
└── 扩展类 (高级功能)
    ├── More/Module - 模块管理
    ├── More/Appearance - 界面设置
    └── More/License - 许可管理

托盘菜单导航：
├── 状态显示 (当前状态)
├── 快速切换 (常用操作)
│   ├── 出站模式切换
│   ├── 代理服务器选择
│   └── 配置文件切换
└── 功能开关 (能力控制)
    ├── 增强模式
    ├── 系统代理
    └── 其他功能
```

### **数据流向分析**
```
配置数据流：
Profile → Policy → Rule → 运行时引擎 → 网络处理

监控数据流：
网络活动 → 数据收集 → Activity/Process/Dashboard → 用户界面

用户操作流：
托盘快速操作 → 主窗口详细配置 → 仪表板数据分析 → 问题排查调试
```

### **状态管理模式**
```
全局状态：
├── 代理启用状态 (开/关)
├── 出站模式 (直连/代理/规则)
├── 当前配置文件
└── 系统代理状态

实时状态：
├── 网络连接状态
├── 流量统计数据
├── 代理服务器健康状态
└── 应用程序网络活动

配置状态：
├── 策略组配置
├── 规则集配置
├── 模块启用状态
└── 功能开关状态
```
